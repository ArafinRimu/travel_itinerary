# travel_itinerary is a dynamic travel planning platform that offers personalized trip management and booking features. The frontend is built with React using Vite to ensure an interactive and responsive user experience. On the backend, APIs are implemented with Node.js and Express.js, while MongoDB is used for data persistence. The application integrates essential modules such as user authentication, blog posts, cart management, and real-time notifications to enhance functionality and user engagement. The tech stack includes React, Node.js, Express.js, MongoDB, and Vite.
